# BMProjects - Полный портфель продуктов

Этот документ содержит исчерпывающий список всех выпущенных и активных проектов, разработанных, поддерживаемых или настроенных студией BMProjects, распределенных по типам проектов.

---

## Модпаки (Сборки)

| Название проекта | Поддерживаемая платформа / Версия | Описание |
| :--- | :--- | :--- |
| **Melody of Utopia** | Minecraft 1.21.1 NeoForge | Флагманский проект (Скоро). Полная пересборка привычного игрового опыта с уникальными механиками, создававшимися на протяжении более 10 месяцев. |
| **Endless Rise: Remastered** | Minecraft 1.16.5 Forge | Высокотехнологичная сборка с эволюционной прогрессией от первобытных эпох до космоса. Включает более 600 квестов и кастомные древа технологий. Заняла Топ-1 в категории "Hidden Gem". |
| **Gloomy Rise** | Minecraft 1.16.5 Forge | Экспертный техномагический хардкорный и RPG модпак, включающий более 550 квестов и более 500 измененных рецептов. |
| **Sacred Logistics** | Minecraft 1.19.2 Forge | Прогрессивная сборка с упором на автоматизацию и логистику, включающая более 650 квестов и более 300 кастомных измененных рецептов. |
| **Technology Valley** | Minecraft 1.21.1 NeoForge | Модпак с продвинутой интеграцией технологий в сочетании с магией, RPG-системами и адаптированными кастомными механиками. |
| **Timeless Magic** | Minecraft 1.20.1 Fabric | Магический модпак с упором на прогрессию, включающий квестовые линии, измененные рецепты и уникальные магические механики. |
| **Classic Remake** | Minecraft 1.19.2 Forge | Улучшенная Vanilla+ сборка, разработанная для модернизации стандартного геймплея с помощью новых предметов и оптимизированного контента. |
| **MagicalLights** | Minecraft 1.16.5 Forge | Магическо-приключенческий пак, использующий кастомные рецепты и уникальные сюжетные квестовые механики. |
| **Fools Rise** | Minecraft 1.21.1 NeoForge | Сатирический сезонный приключенческий модпак с уникальными циклами прогрессии и кастомными механиками. |

## Кастомные моды и технические аддоны

| Название мода | Категория | Описание |
| :--- | :--- | :--- |
| **Endless Rise Tweaks** | Утилиты / Разное (QoL) | Системные исправления, основные оптимизации и модификации скриптов, созданные специально для модпака Endless Rise: Remastered. |
| **Sacred Logistics Tweaks** | Основной аддон / API | Специализированный мод-аддон, предоставляющий кастомные таблицы интеграции, управление ресурсами и скрипты генерации мира для Sacred Logistics. |
| **Technology Valley Tweaks** | Утилиты / API | Кастомный слой технической интеграции и расширения API, созданные специально для управления механиками в Technology Valley. |
| **Fools Rise Tweaks** | Прогрессия / Генерация мира | Компактный мод для внедрения скриптов и структур, управляющий элементами прогрессии в сезонном проекте Fools Rise. |
| **SeasonHud Tweaks** | Косметический / Карты | Аддон, предоставляющий улучшения интерфейса, переопределение конфигураций и косметические оптимизации для мода SeasonHud. |
| **Polymorph Artisan Worktables** | Утилиты / Совместимость | Мод технической совместимости, обеспечивающий полную поддержку выбора предметов для верстаков Artisan Worktables в среде 1.16.5. |
| **Gravestone Curios API Compat Backport** | Утилиты / Совместимость | Бэкпортированный технический мост, устанавливающий структурную совместимость между Gravestone Mod и Curios API для версии 1.16.5. |
